Installing Power Polygon

It is easier than it looks like.

First thing, clone or download Power Polygon: 
    $ git clone https://github.com/braziljs/power-polygon.git
If you want, you can use nodejs to enable many more features(optional, but recommended): 
    $ cd power-polygon
    $ npm install
    # You will be asked to create a token
    #(your token is a keyword you will use later)
    $ sudo node run.js
    # this starts the server, listening to port 8081
DONE: You are good to go!
Access in your browser: http://localhost/ (in case you just unziped/cloned it under another server, like apache).
Or, if you installed using node(recommended), access: http://localhost:8081/

Create your talk

Creating your talk

Go to the Power Polygon directory, then:
1) Go to the "talks" directory
2) Create a folder inside it with the name my-talk.
3) Inside your new talk's folder, create an "index.html"
4) Paste this as the content of the index.html: 
<!DOCTYPE html>
<html>
    <head>
        <title>My presentation</title>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">

        <script src="../../ppw/_scripts/jquery-1.8.2.min.js"></script>
        <script src="../../ppw/ppw.js"></script>
        <!-- you can load other scripts, plugins or css files here -->

    </head>
    <body>

        <section id="slide-0">
            <h1>My first slide!</h1>
        </section>

        <script>
            PPW.init(); // pay attention here
        </script>
    </body>
</html>
5) Access http://localhost:8081/my-talk/ to see your talk.
* if not using nodejs, it will be like http://localhost/power-polygon/talks/my-talk/

Creating your talk

Great! You got your talk running!
As you may have noticed, add <section> elements with ID, for each slide you want to add to the presentation.

BUT, if you want your slides more organized, you can replace the line with the comment "pay attention here" by this: 
PPW.init({
    slides: [
        { id: 'slide-0', type: 'opening' },
        { id: 'about-me' }, // by default, type is "content"
        { id: 'section-title', type: 'section' }, // section slides are special
        { id: 'slide-example' },
        { id: 'thanks', type: 'closing' } // the last slide
    ]
});

Creating your talk

You can add properties to the object passed to PPW.init().
The available parameters are: https://github.com/braziljs/power-polygon/wiki/Create your talk#wiki-Full definition

For example, you can change the theme or transition for your talk like this: 
PPW.init({
    theme: 'thm-red-n-black',
    transition: 'trans-3d-3',
    slides: ...
});
But you can save a valid json(careful with the mandatory " instead of ' ) in another file, and specify it here as follows: 
PPW.init("talk_definition.json");

Slide Features

Now, you can add more interaction to your slides, like actions, animations, support to
different languages, use the camera, different profiles, etc...
Let's see how...

Tools (toolbar)

You can, by anytime, go with your mouse to the top-left corner of the screen and see the toolbar.
Try it now!

You can Use it to:
  • Search a text on the presentation
  • Jump to another slide
  • See slides thumbnails
  • See printable version
  • Increase/decrease font size
  • Toggle camera
  • Toggle fullscreen
  • Open the presentation toolbox
  • Enable remote control
  • Show the Power Polygon settings

Slide Actions

You can add actions to your slides, so, going next or previous will not go to the next slide, but execute your action.
To do that, add INSIDE your slide(inside your section, or in your slide file, in case it is an external slide).
PPW.addAction( function (){ alert(0); } );
But you can have a does, and undo functions for your action: 
PPW.addAction({
    does: function(){ alert("doing something"); },
    undo: function(){ alert("undo that!"); },
});
You can add as many actions for each of your slides, as you want.

You can also add a timing property(in milliseconds).
It will make your action run on schedule, instead of on click/next event. 
PPW.addAction({
    // will execute the 'does' function automatically, after 1 second
    timing: 1000, // you can use the word 'auto', as equivalent to 0
    does: function(){ alert("doing something"); },
    undo: function(){ alert("undo that!"); },
});
Try going back and forward. These animations are in the same slide.

Slide Animations

Now that you know how to add actions to your slides, it's time to use optimized animations.
Once jQuery is loaded to your presentation, you can, of course, use it to animate elements. Although, with PPW.animate you can perform pre-defined animations made in CSS(therefore, probably with a better performance).
PPW.animate("selector", "animation", {options: values});
Options are optional. It is an object that may contain:
  • count: the number of repetitions or the world 'infinite'
  • duration: a string with the duration in seconds or milliseconds(CSS format)
  • delay: time delay, also in CSS format
  • onstart: function to be triggered when the animation starts
  • onend: function to be triggered when the animation ends

Slide Animations

Example: 
PPW.addAction({
    timing: 'auto',
    does: function(){
              PPW.animate("#slide-animations-eg",
                  "pulse",
                  {
                      count: 'infinite',
                      duration: '500ms',
                      delay: '2s',
                      onstart: function(){ console.log(0); },
                      onend: function(){ console.log(1); }
                  }
              );
          }
});
You can see the list of available animations by typing PPW.animations.

Languages

You can create your talks with as many languages as you need. Você pode criar suas palestras com quantos idiomas precisar.
<span class="LANG-EN">Text in English.</span>
<span class="LANG-PT">Texto em Português.</span>
Go to the top-left corner of the screen, open the settings and change the language selectbox.

Profiling your talk

You can profile your talk by setting the default profile in the init method, and defining the slides their profile.
PPW.init({
    profile: 'defaultProfile', // you name it
    // other settings you may want to use
    slides: [
        { id: 'slide-1' },// default profile
        { id: 'slide-2', profile: 'other' },
        { id: 'slide-3' },// default profile
    ]
});
Once you do it, you can go to settings, using the toolbar in the top-left corner of the screen, and change it.

Plugins

Plugins

Power Polygon can be extended so you can use plugins.
To do that, all you gotta do is to include your chosen plugin script right after Power-polygon's script.
Like this: 
<script src="../../ppw/_scripts/jquery-1.8.2.min.js"></script>
<script src="../../ppw/ppw.js"></script>

<script src="../../ppw/_addons/syntaxhighlighter/sh.js"></script>
<script src="../../ppw/_addons/chartjs/chart.js"></script>
You can find the list of available addons in the _addons directory.

Remote Control

Remote Control

You can control your presentation via your cellular phone or tablet.
You can also draw, point or focus, with this remote control.
To use the remote control, follow these steps:
  1. Start the node service from you node instalations of Power Polygon. 
    $ sudo node run.js
  2. Access your talk via your IP address in port 8081 in both you computer and cellphone/tablet.
    Something like: http://192.168.0.11:8081/
  3. In your computer, click on the icon to open the talk you want to present to your audience.
  4. In your computer, connected to the projector, enable the remote control
    (the little antenna in the toolbar, on the top-left corner of the screen)
    *verify if the antenna was not already on...it tries to keep the last state
  5. In your controller(the cellphone or tablet), instead of opening your talk, click on the "padlock" icon.
    Type your token, and the remote control icon will be enabled, click on the remote control icon for the talk you want to control.
