Syntax Highlight

All the source coude examples in this presentation use this addon.
To use it, load the plugin after including the 'ppw.js' script: 
<script src="../../ppw/ppw.js"></script>
<script src="../../ppw/_addons/syntaxhighlighter/sh.js"></script>
Then, you create a pre element with a class starting with sh_ and the name of the language.
Like this: 
<pre class="sh_html">your code here</pre>
<pre class="sh_php">your code here</pre>
<pre class="sh_javascript">your code here</pre>
